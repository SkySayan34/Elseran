# Amulette de sombre éclat

type: Objet merveilleux
rarete: commun
necessite_lien: Yes
source: Xanathar´s Guide to Everything
description: Cette amulette est fabriquée à partir d'un éclat résistant de matériau extraplanaire provenant du royaume de votre patron d'occultiste. Tant que vous le portez, vous bénéficiez des avantages suivants :
Vous pouvez utiliser l'amulette comme focaliseur d'incantation pour vos sorts d'occultiste.
Vous pouvez essayer de lancer un sort mineur que vous ne connaissez pas. Celui-ci doit figurer sur la liste des sorts d'occultiste et vous devez effectuer un jet d'Intelligence (Arcanes) DD 10. En cas de réussite, vous lancez le sort. En cas d'échec, le sort échoue également et l'action utilisée pour lancer le sort est perdue. Dans les deux cas, vous ne pourrez plus utiliser cette propriété avant d'avoir terminé un repos long.
