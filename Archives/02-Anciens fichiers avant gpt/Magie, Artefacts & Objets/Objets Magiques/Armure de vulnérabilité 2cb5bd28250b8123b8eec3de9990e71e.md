# Armure de vulnérabilité

type: Armure
sous_type: harnois
rarete: rare
necessite_lien: Yes
source: Dungeon Master´s Guide (SRD)
description: Tant que vous portez cette armure, vous avez une résistance contre l'un des types de dégâts suivants : contondant, perforant ou tranchant. Le MD choisit le type ou le détermine aléatoirement.
Malédiction
. Cette armure est maudite, un fait qui n'est révélé que lorsqu'un sort d'
identification
est lancé sur l'armure ou si vous vous liez à elle. Se lier à l'armure vous maudit jusqu'à ce que vous soyez ciblé par le sort
délivrance des malédictions
ou une magie similaire ; retirer l'armure ne permet pas de mettre un terme à la malédiction. Tant que vous êtes maudit, vous avez une vulnérabilité face aux deux autres types de dégâts associés à cette armure (pas celui contre lequel l'armure vous confère une résistance).
