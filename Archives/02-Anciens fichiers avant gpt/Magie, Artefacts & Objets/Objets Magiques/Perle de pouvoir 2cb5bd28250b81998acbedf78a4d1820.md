# Perle de pouvoir

type: Objet merveilleux
rarete: peu commun
necessite_lien: Yes
source: Dungeon Master´s Guide (SRD)
description: Tant que cette perle est sur vous, vous pouvez utiliser une action pour prononcer son mot de commande et récupérer un emplacement de sort dépensé. Si l'emplacement de sort était de niveau 4 ou supérieur, le nouvel emplacement est de niveau 3. Une fois que vous avez utilisé la perle, elle ne peut être réutilisée avant la prochaine aube.
